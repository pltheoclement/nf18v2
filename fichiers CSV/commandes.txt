Pour importer la base de données:

    \i sql_veto.sql
    
Insérer les données dans le tableau des espèces:

     \copy espèces (nom) FROM 'especes.csv' WITH CSV DELIMITER ';' QUOTE '"'
     
Insérer les données dans le tableau des personnel:

    \copy personnel FROM 'personnel.csv' WITH DELIMITER AS ';' CSV HEADER
